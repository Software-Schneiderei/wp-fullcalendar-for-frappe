Adds a &quot;Fullcalendar&quot; block to the editor which fetches Events from an API

== Description ==

Adds Fullcalendar block to the Gutenberg editor, which fetches events from the URL specified in the block options.
This plugin is developed for a German speaking customer, so currently no i18n has taken place.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/fullcalendar-for-frappe` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
